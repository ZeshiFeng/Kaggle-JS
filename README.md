# Kaggle-JS
Jane Street Real-Time Market Data Forecasting Competition

Download the competition data from:
https://www.kaggle.com/competitions/jane-street-real-time-market-data-forecasting/data

Run the files nn_train.ipynb, ridge_train.ipynb, xgb_train.ipynb, and tabm_train.ipynb to obtain model weights.
Run the ensemble.ipynb file.

Note: At least one GPU and a configuration with 64 GB or more memory are required.
